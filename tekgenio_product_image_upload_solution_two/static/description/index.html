Features

  • Authentication- Authenticate using your Odoo Database credentials to connect with odoo.
  • Efficient Folder Selection-Users can make root directory anywhere in the system, which containing folders corresponding to product barcode numbers. This folders contains main image and the extra images of the Odoo products as per the mentioned format in the screenshots section.
  • Smart Barcode Verification-It cross-checks the names of selected folders to ensure they match the product barcode numbers in Odoo and end of the operation you can see the log reprot in odoo.
  • Sync Images to Upload In Odoo- The application trigger the sync operation to upload images in odoo based on the modified date timestamp.
