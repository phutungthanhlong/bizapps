Import images from Zip Odoo app helps to import/add bulk images from zip file for product, partner or customer and employee. If you have many images and want to update all images then our module will help you to import/add that very easily. Manage all images in one zip file and import it, it will auto-update all images. Also this module will be useful to update images for the product, partner/customer and employee. you can import images by name, reference, barcode and IDS depending on product, partner and employee.

Configuration for Import Images from Zip File

Goto Settings -> Users & Companies -> Users In that tick "Import Images From Zip File" group.


Import Images From Zip File Menu in Sales

You can select import images from zip file menu in sales.


Import Images From Zip File Wizard

1.Import Images For :

You have option for import images from zip file for Product, Partner or Employee.

2.Import Product By :

You have also option to import products by it's Name, Internal Reference(code), Barcode or ID.

If you want to import product images while importing images from zip file then you have to select import with "Name".

3.Product Model :

User can see products with images.


You have also option to import images from zip file in that product model by product template or product variants.

User can download sample zip file to import product images by clicking "ZIP FILE" button on wizard.
